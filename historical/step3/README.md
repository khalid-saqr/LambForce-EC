# Historical Step 3 checksum manifest

`65d4f7252b09b17d466ec13096335129f80465b6`.

It was moved out of the repository root because later Phase 0 and readiness changes intentionally
modify the listed files. It must not be used to validate the current package. Current claim-bearing
artifacts use per-record, source-registry, configuration, and result checksums instead.
